# snake-stream-# Snake Stream Overlay 🎮

A TikTok LIVE interactive snake game powered by Tikfinity.

## Features

* AI-controlled snake
* Gift-triggered events
* Boss fights
* OBS-ready overlay

## Setup

1. Run Tikfinity
2. Enable WebSocket (ws://localhost:21213)
3. Open OBS → Browser Source
4. Paste your GitHub Pages URL

## Controls

* Gifts = gameplay effects
* Galaxy = Boss Fight
